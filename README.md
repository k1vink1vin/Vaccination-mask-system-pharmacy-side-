此程式開發目的是方便藥局端同時能夠監控藥局的剩餘量，又同時能夠過卡。

## 使用方法
-   執行mask.exe

跟原本過卡相同，只是增加了旁邊的即時資訊欄，方便藥局進行數量的監控。
此數量是來自於健保署提供的即時資料。 

系統每30秒自動更新一次，亦可手動更新。
